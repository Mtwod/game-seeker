# Game Seeker

Welcome to the repo of "Game Seeker"! You want to find a game by platform, similar genre or the most wanted release? You came to the right place!

## Introduction

This project is based on [The Hacking Project (THP)](https://www.thehackingproject.org/) formation's project ["The Hyper Progame"](https://github.com/Mtwod/72_hours_PLS).

Its purpose was to code a website with pure JavaScript, with data fetching from the RAWG API to display two pages:

- A list of games, with a specific order and conditions

- Details of a game

## Objectives

The purpose of this personal project is to remake this project, by using the most of what I learned since then on React.
