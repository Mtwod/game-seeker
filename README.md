# Game Seeker

This project is based on [The Hacking Project (THP)](https://www.thehackingproject.org/) formation's project "The Hyper Progame".

Its purpose was to code a website with pure JavaScript, with fetching to the RAWG API to display two pages:

- A list of games, with a specific order and conditions

- Details of a game

The purpose of this personal project is to use the most of what I learned since then.
